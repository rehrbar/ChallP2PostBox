## Azure Cloud Service

1. In the hub navigation, click on "Cloud service (classic)"
2. Enter a valid DNS name, for example "tapbox"
3. Select your subscription the service uses.
4. Select the resource group you created earlier, for example "ChallengeProj".
5. Select the location nearest to you.
6. Click on "Create".
